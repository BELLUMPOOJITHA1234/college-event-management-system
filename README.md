## 📖 Overview

The **College Event Management System** is a web-based platform designed to manage and showcase different types of college events such as **technical, cultural, sports, and club activities**.

This project provides a structured interface where students can easily explore upcoming events while maintaining a simple and efficient system for organizing event information.

---

## 🚀 Features

✨ Explore multiple categories of college events
🎭 Dedicated pages for **Cultural Events**
💻 Sections for **Technical Competitions**
🏆 Sports event listings
👥 Club activities and campus programs
🔐 User **Login & Registration system**
⚡ Fast and lightweight web interface

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/BELLUMPOOJITHA1234/college-event-management-system.git
```

### 2️⃣ Navigate to the project folder

```bash
cd college-event-management-system
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Create a `.env.local` file

Add your Gemini API key:

```
GEMINI_API_KEY=your_api_key_here
```

### 5️⃣ Run the development server

```bash
npm run dev
```

### 6️⃣ Open the application

```
http://localhost:3000
```
