# College Event Management System

A web-based application for managing and showcasing college events including technical, cultural, and sports activities.

## Overview

This project provides a structured platform where users can explore events and administrators can manage event information efficiently.

## Tech Stack

- Node.js
- JavaScript
- HTML & CSS
- Gemini API

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm

### Installation

Clone the repository:

git clone https://github.com/your-username/your-repo-name.git

Navigate to the project directory:

cd your-repo-name

Install dependencies:

npm install

Create a `.env.local` file and add your API key:

GEMINI_API_KEY=your_api_key_here

Start the development server:

npm run dev

The application will run on:

http://localhost:3000
